Programming Problem Repository

Lonnie Hansen, Leah Smith, Sunny To, San Tran, Paul Inventado
Github repository: https://github.com/ilxl-ppr

Development of a comprehensive repository that contains programming problems with automated tests and graders to provide students with immediate feedack. This repository is being designed as a lab-excercise resource for cs professors teaching core coding classes.

Although this repository is currently limited to C++ problems the goal is to eventually enhance the repository to support each problem in multiple languages.

Development of supplemental programming guides and tutorials for CSUF students.

Programming Problem Repository Web Application

Ruby Abutaleb, Dayna Anderson, Andrew Gomez, Evan Purpura, Paul Invendato
Github repository: https://github.com/ILXL/ppr-web

Development of a web-based interface for the Programming Problem Repository to help instructors manage its content, deploy programming problems in class, and facilitate grading.

Later functionality goals include the generation of gitHub classroom projects and the ability to populate grade results back into Titanium.
